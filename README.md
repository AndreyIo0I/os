## (0) Перевод из Мили в Мура.
```
> node convert
```
```
> node convert mealy.txt
```
```
> node convert moore.txt
```
Тк из мура в мили перевести легко, при чтении автомат сразу преобразовывается в мили,
чтобы не было проблем с двумя вариантами.
